Bem-vindo ao Sistema de Reajuste Salarial

Para calcular seu reajuste salarial, acesse a URL no seguinte formato:

https://seu-projeto.vercel.app/?idade=30&sexo=F&salario_base=1700&anoContratacao=2014&matricula=12345

Parâmetros obrigatórios:

  • idade: Idade atual (ex: 30)
  • sexo: "M" para masculino, "F" para feminino
  • salario_base: Salário base atual (ex: 1700)
  • anoContratacao: Ano em que foi contratado (ex: 2014)
  • matricula: Número da matrícula funcional

Exemplos prontos para testar:

https://seu-projeto.vercel.app/?idade=28&sexo=M&salario_base=2500&anoContratacao=2020&matricula=1001
https://seu-projeto.vercel.app/?idade=42&sexo=F&salario_base=3100&anoContratacao=2010&matricula=1002
