Como calcular seu reajuste salarial

Para calcular o reajuste, acesse a URL no formato abaixo:

http://localhost:3000/?idade=30&sexo=F&salario_base=1700&anoContratacao=2014&matricula=12345

Parâmetros obrigatórios:

  • idade → Idade atual (ex: 30)
  • sexo → "M" para masculino, "F" para feminino
  • salario_base → Salário base atual (ex: 1700)
  • anoContratacao → Ano em que você foi contratado (ex: 2014)
  • matricula → Número da matrícula funcional
